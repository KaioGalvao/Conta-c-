# Conta Bancária

Aplicação console para criar Contas.

## Funcionalidades

- Cria Conta.
- Realiza o depósito de um valor.
- Realiza o saque de um valor.
- Realiza transferência bancária.

## Tecnologias utilizadas

- dotnet 6.0

## Como executar

Realize o clone do repositório e execute o comando:
```
dotnet run
```
